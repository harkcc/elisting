# GIF / Motion Components

These components provide stable low-risk motion. They can be embedded into a generated/composited image or into eMAG HTML through an image tag.

Stable v1 components:

- `shine_sweep`: subtle highlight across a brand/trust/proof module.
- `step_highlight`: numbered steps light up in sequence.

Experimental and not default:

- people micro-motion
- product rotation
- product edge glow
- particle field
- breathing border

Rules:

- The static first frame must still communicate the module.
- Product shape must not deform.
- Motion must support proof, sequence, or brand familiarity.
- Motion must not reduce mobile readability.
