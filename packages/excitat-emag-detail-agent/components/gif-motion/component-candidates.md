## Approval Criteria

- Motion supports proof, order, or brand identity.
- Static fallback still works.
- No product deformation.
- No fake service, review, warranty, return, or marketplace claim.
- Fits a 390px mobile preview.
- Can be embedded as an HTML snippet now and exported to GIF/WebP later.
