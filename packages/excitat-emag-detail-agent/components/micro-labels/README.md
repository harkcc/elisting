# Micro Labels

Micro labels are small reusable ordering and emphasis components. They are separate from brand identity components.

They answer:

- What is the order?
- Which item is important?
- What type of proof is this?
- Which marker, rank, spec, or module family is active?

Initial candidates:

- `e2_marker_chip`
- `rank_number_strip`
- `proof_check_chip`
- `spec_icon_marker`
- `module_section_marker`
- `corner_status_badge`

Rules:

- Micro labels may be embedded into composited images or eMAG-safe HTML.
- Labels must not invent platform claims, review facts, or service promises.
- Text must remain readable at small sizes.
- Animated variants must have static fallbacks.
