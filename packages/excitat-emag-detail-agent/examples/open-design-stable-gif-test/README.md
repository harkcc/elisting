# Open Design Stable GIF Test

This is a local HTML design test for applying the stable GIF component library
with Open Design rules.

It is not the final eMAG paste format. Final eMAG output should still export
GIF/PNG media and embed it with simple `p/img` snippets.

Open:

```text
examples/open-design-stable-gif-test/index.html
```

Rules demonstrated:

- one premium black-gold hero only
- white/off-white micro component cards for brand label, corner badge, rank strip, and pulse dot
- slow luxury motion for shine and glow
- small motion after text/module intent is stable
- no fake proof, service promise, platform badge, or noisy product-image background
