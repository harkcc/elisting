# Prompt Pattern: Brand Trust Banner

Use for premium brand-trust backgrounds or scenes.

Template:

```text
Create a premium brand-trust ecommerce banner background.
Mood: reliable, warm, refined, not luxury fashion.
Subject: [people/package/service_context] with no visible brand or marketplace marks.
Composition: reserve [left/right] area for deterministic EXCITAT / X-City wordmark and up to three service/evidence cues.
No readable text, no logo, no discount badge, no warranty badge, no marketplace mark.
Avoid unsupported platform guarantees, fake review visuals, fake shipping or return promises, and crowded layouts.
```
