# Prompt Pattern: Repair

Use after Visual QA identifies a specific failure.

Template:

```text
Repair the candidate image for [image_job_id].
Keep: [approved_elements].
Fix: [specific_failure].
Preserve: composition role, reserved overlay area, brand tone, and product slot.
Do not add readable text, logos, fake badges, review stars, warranty claims, shipping or return promises, or marketplace marks.
Avoid distorted product shapes, extra small labels, and visual clutter.
```
