# Prompt Pattern: Proof Board

Use for premium proof boards only when deterministic composition needs a generated scene/background.

Template:

```text
Create a clean proof-board background for [proof_topic].
Leave clear areas for deterministic icons, product crop, and short proof text.
Style should match [brand_tone] and [reference_banner_pattern].
No readable text, no numbers, no logos, no fake badges, no review stars.
Avoid visual clutter and decorative elements that compete with the proof.
```
