# Prompt Pattern: Background Only

Use when the model should create only atmosphere and layout space.

Template:

```text
Create a no-text ecommerce banner background for [product_category].
Canvas: [canvas_size].
Mood: [premium / warm / technical / clean / family-friendly].
Composition: reserve [reserved_zone] as clean empty space for deterministic overlay.
No products, no readable text, no logos, no badges, no marketplace marks.
Avoid clutter, fake UI labels, fake packaging, fake certifications, and overfilled decoration.
```
