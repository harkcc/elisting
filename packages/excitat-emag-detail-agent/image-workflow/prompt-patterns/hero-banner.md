# Prompt Pattern: Hero Banner

Use for premium first-screen banners.

Template:

```text
Create a premium ecommerce detail-page hero background for [product_category].
Use the reference banner for composition rhythm, color mood, and lighting direction, without copying logos, text, people, or marketplace-specific assets.
Canvas: [canvas_size].
Scene goal: communicate [core_result] and [buyer_pain_removed].
Composition: reserve [left/right/top] clean space for deterministic product, brand, and text overlay.
Product policy: do not redraw precise product details; leave a natural product slot for a real product cutout.
Text policy: no readable text, no logo, no fake badge, no marketplace mark.
Mood: [brand_tone].
Avoid: no readable text, no fake badge, no marketplace logo, no warranty claim, no shipping or return promise, no distorted product, no crowded layout.
```
