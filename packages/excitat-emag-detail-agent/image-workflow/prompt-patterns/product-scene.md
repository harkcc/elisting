# Prompt Pattern: Product Scene

Use for product-context banners and proof scenes.

Template:

```text
Create a clean ecommerce scene for [product_category] that shows [usage_context].
Use reference images only for composition, material mood, and lighting.
Reserve [slot] for deterministic insertion of the real product image.
The scene should support this buyer result: [core_result].
No readable text, no logo, no badge, no fake guarantee, no marketplace mark.
Avoid distorted products, overfilled layouts, tiny labels, and copied official marketplace assets.
```
